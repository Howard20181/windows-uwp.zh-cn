# <a name="getting-started-with-uwp-for-ios-developers"></a>适用于 iOS 开发人员的 UWP 入门


如果你是一名负责开发 Windows 10 的 iOS 开发人员，则可以开始使用这些文档。 他们将向你介绍你在开始编写应用时需要了解的一些概念，并介绍如何在 Microsoft Store 上发布工作。

在本部分中，我们将逐渐开始并查看如何使用 Microsoft Visual Studio 和 C# 编程语言创建简单应用，特别是此过程会如何随着当前使用工具的变化而变化。 （对 C# 不感兴趣？ 不用担心，我们会提供其他编程语言和工具，并且将在[入门：选择一种编程语言](getting-started-choosing-a-programming-language.md)中介绍它们）。

Windows 10 引入了一个新平台，用于在台式机、便携式计算机、平板电脑、手机设备等上创建富有吸引力的应用程序。 由于通用 Windows 平台 (UWP) 应用提供了许多独特的功能，因此，直接移植你的 iOS 应用可能会错过这些功能。 因此，我们鼓励你尝试新控件和新功能，查看它们如何使你作为开发人员的生活更轻松，以及帮助你开发新应用。

不要单单移植应用，而是要**重新构建**应用，并且利用新功能和新设备，这是底线。 不要为最小的普通分母进行结算，而是创建使用独特 Windows 10 功能的丰富体验，如动态磁贴、通知和与 Cortana 交互。

若要开始使用这些演练，将需要一台安装了 Windows 10 和 Microsoft Visual Studio 的计算机。 可以从[适用于 UWP 应用的开发人员下载](https://developer.microsoft.com/windows/downloads)下载这些内容。 没有 PC？ 不要担心，你可以使用 Mac：请参阅[在 Mac 上安装 Windows 和开发人员工具](setting-up-your-mac-with-windows-10.md)。
