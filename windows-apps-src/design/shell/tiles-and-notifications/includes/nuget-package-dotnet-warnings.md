> [!IMPORTANT]
> 仍使用 packages.config .NET Framework 桌面应用必须迁移到 PackageReference，否则不会正确引用 Windows 10 Sdk。 在项目中，右键单击 "引用"，然后单击 "将 packages.config 迁移到 PackageReference"。
> 
> .NET Core 3.0 WPF 应用必须更新到 .NET Core 3.1，否则将不会存在 Api。
